```typescript
import { S3Client, PutObjectCommand, DeleteObjectCommand } from '@aws-sdk/client-s3';

const r2Client = new S3Client({
  region: 'auto',
  endpoint: `https://${process.env['R2_ACCOUNT_ID']}.r2.cloudflarestorage.com`,
  credentials: {
    accessKeyId: process.env['R2_ACCESS_KEY_ID']!,
    secretAccessKey: process.env['R2_SECRET_ACCESS_KEY']!,
  },
});

export async function uploadToR2(key: string, body: Buffer, contentType: string): Promise<void> {
  await r2Client.send(new PutObjectCommand({
    Bucket: process.env['R2_BUCKET']!,
    Key: key,
    Body: body,
    ContentType: contentType,
  }));
}

export async function deleteFromR2(key: string): Promise<void> {
  await r2Client.send(new DeleteObjectCommand({
    Bucket: process.env['R2_BUCKET']!,
    Key: key,
  }));
}
```

**1. Create `apps/backend/src/workers/voice-processor.ts`:**

Export a singleton `boss` instance and `initBoss()` function. The worker processes `voice-process` jobs with this flow:

```typescript
import PgBoss from 'pg-boss';
import OpenAI from 'openai';
import { query } from '../db/connection.js';
import { deleteFromR2 } from '../services/r2.js';
import type { VoiceJobData, AILineItem } from '../types/voice.js';
```

**`initBoss()`**: Creates PgBoss with `process.env['DATABASE_URL']!`, adds error handler (`console.error`), calls `await boss.start()`, then registers worker: `await boss.work<VoiceJobData>('voice-process', { teamSize: 2 }, processVoiceJob)`.

**`processVoiceJob(job)`**: Implements the full pipeline:

a) **Fetch audio from R2** — use `GetObjectCommand` to get the audio buffer from R2 using `job.data.r2Key`. Convert the stream Body to a Buffer. Create a `File` object from it for the OpenAI API.

b) **Whisper transcription**: 
```typescript
const openai = new OpenAI({ apiKey: process.env['OPENAI_API_KEY'] });
const transcription = await openai.audio.transcriptions.create({
  model: 'whisper-1',
  file: audioFile,
});
const transcript = transcription.text;
```

c) **Delete audio from R2 immediately** (D-08, PII): `await deleteFromR2(job.data.r2Key);`

d) **Fetch contractor's active catalog**:
```sql
SELECT id, name, unit FROM catalog_items WHERE contractor_id = $1 AND is_archived = false
```

e) **GPT-4o function calling** (per D-07, VOICE-05):
System prompt: "You are a trade quoting assistant. Map the job description to items from this catalog only. Catalog: {JSON of catalog items with id, name, unit}. Return ONLY items that exist in this catalog. Use the exact catalog IDs. Assign a confidence score (0 to 1) indicating how well the transcript matches each item."

Tool definition:
```typescript
tools: [{
  type: 'function',
  function: {
    name: 'create_quote_items',
    description: 'Return the line items for this quote',
    parameters: {
      type: 'object',
      properties: {
        items: {
          type: 'array',
          items: {
            type: 'object',
            properties: {
              catalogItemId: { type: 'string', description: 'ID from the provided catalog' },
              quantity: { type: 'integer', minimum: 1 },
              confidence: { type: 'number', minimum: 0, maximum: 1, description: 'How confident this item belongs in the quote' },
            },
            required: ['catalogItemId', 'quantity', 'confidence'],
          },
        },
      },
      required: ['items'],
    },
  },
}],
tool_choice: { type: 'function', function: { name: 'create_quote_items' } },
```

f) **Parse response**: Extract `items` array from `completion.choices[0]?.message.tool_calls?.[0]?.function.arguments`. Parse as `{ items: AILineItem[] }`.

g) **Validate catalog IDs** (VOICE-06):
```sql
SELECT id, name, unit_price_cents FROM catalog_items WHERE contractor_id = $1 AND id = ANY($2) AND is_archived = false
```
Filter `items` to only those with IDs in the valid set. For each valid item, look up `name` and `unit_price_cents` from the DB result.

h) **Build line items JSON**: For each validated item, create `{ catalogItemId, name, quantity, unitPriceCents, confidence }` from the DB lookup + GPT response. Serialize as JSON.

i) **Calculate total**: Sum of `quantity * unitPriceCents` for all validated items.

j) **Write draft line items and update quote status**: In a single transaction:
- Create a row in `drafts` table (the draft doesn't exist in Postgres yet for voice quotes — this is a mobile-initiated concept, but store as JSON for the polling response):
  ```sql
  INSERT INTO quote_line_items (quote_id, name, quantity, unit_price_cents) VALUES ($1, $2, $3, $4) RETURNING id
  ```
  Insert one row per validated line item.
- Update the quote:
  ```sql
  UPDATE quotes SET status = 'draft_local', total_cents = $1 WHERE id = $2
  ```

k) **Store completion status**: Update `voice_jobs` table (pg-boss handles this internally — the status endpoint queries pg-boss job state).

**Error handling**: Wrap entire `processVoiceJob` in try/catch. On error, update quote status to `failed_send` (or keep `ai_processing` and let pg-boss retry). Log error with `console.error`.

Export `boss` and `initBoss`.

**2. Create `apps/backend/src/routes/voice.ts`:**

```typescript
import { Router, Request, Response } from 'express';
import multer from 'multer';
import { v4 as uuidv4 } from 'uuid';
import { authenticateToken } from '../middleware/auth.js';
import { uploadToR2 } from '../services/r2.js';
import { boss } from '../workers/voice-processor.js';
import { query } from '../db/connection.js';
import type { VoiceStatusResponse } from '../types/voice.js';
```

`POST /upload` (authenticateToken, multer memoryStorage with 25MB limit):
- Extract `contractorId` from `req.contractor!.contractorId`
- Validate `req.file` exists, return 400 if not
- Generate R2 key: `audio/${contractorId}/${uuidv4()}.m4a`
- Create quote with status `ai_processing`: `INSERT INTO quotes (contractor_id, status, total_cents) VALUES ($1, 'ai_processing', 0) RETURNING id`
- Upload to R2: `await uploadToR2(r2Key, req.file.buffer, req.file.mimetype)`
- Send pg-boss job: `const jobId = await boss.send('voice-process', { quoteId, contractorId, r2Key })`
- Update quote with job ID: `UPDATE quotes SET voice_job_id = $1 WHERE id = $2` with params `[jobId, quoteId]`
- Return 202: `{ jobId, quoteId }`

`GET /status/:jobId` (authenticateToken):
- Fetch job from pg-boss: `const job = await boss.getJobById(jobId)`
- If no job: return 404 `{ error: 'Job not found' }`
- Map pg-boss state to response: if `job.state === 'completed'` → `{ status: 'complete', draftId: job.data.quoteId }`, if `job.state === 'failed'` → `{ status: 'failed', error: 'Processing failed' }`, otherwise → `{ status: 'processing' }`
- Additionally verify the quote belongs to the requesting contractor: `SELECT id FROM quotes WHERE voice_job_id = $1 AND contractor_id = $2`

Export `router`.
