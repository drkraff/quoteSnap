Exceptions:
- FAB minimum height: 56px (comfortable for gloved/on-site hands — D-01 locked)

- Flagged LineItemRow (confidence badge visible) minimum height: 72px (D-05 locked — badge is sibling below name text)
- FlatList `contentContainerStyle.paddingBottom`: computed as `tabBarHeight + insets.bottom + fabStackHeight` — prevents last quote row being hidden under FAB stack (D-01 locked)

## Interaction Contracts

### Dual FAB Stack (quotes.tsx)

```
[Voice Quote FAB] — bottom, primary
  position: absolute
  bottom: insets.bottom + tabBarHeight + 16
  right: 16
  width: 56, height: 56, borderRadius: 28
  backgroundColor: colors.accent (#0066cc)
  icon: mic-outline (Ionicons), size 24, color #ffffff
  label: "Voice Quote", 14px, weight 600, color #ffffff
  elevation: 4 (Android) / shadowOpacity 0.25 (iOS)
  accessibilityLabel: "Start voice quote"
  accessibilityRole: "button"

[Manual FAB] — 12px above Voice FAB
  position: absolute
  bottom: insets.bottom + tabBarHeight + 16 + 56 + 12
  right: 16
  width: 56, height: 56, borderRadius: 28
  backgroundColor: colors.secondary (#f5f5f5)
  borderWidth: 1, borderColor: colors.border (#cccccc)
  icon: create-outline (Ionicons), size 24, color colors.mutedText (#666666)
  label: "Manual", 14px, weight 600, color colors.mutedText (#666666)
  elevation: 2 (Android) / shadowOpacity 0.15 (iOS)
  accessibilityLabel: "Create manual quote"
  accessibilityRole: "button"
```

State rules:
- Both FABs always visible — no expand/collapse, no long-press, no speed-dial (D-01 locked)
- While `isCreating` is true on Manual FAB: opacity 0.6, disabled=true
- Voice FAB is never disabled — recording screen handles mic permission state

### Recording Modal (voice-record.tsx)

States:

| State | Mic Button | Waveform | Duration | Bottom CTA |
|-------|-----------|----------|----------|------------|
| idle (pre-record) | Large mic icon, accent fill, 80px circle | Hidden | Hidden | "Tap to record" hint text, mutedText |
| recording | Large stop icon, destructive fill, 80px circle (pulse animation) | Visible, animated | Counting up, HH:MM:SS format, label style, mutedText | "Tap to stop" hint text, mutedText |
| stopped (processing handoff) | Disabled, opacity 0.4 | Hidden | Final duration, static | "Processing..." then auto-dismiss |

- On Stop: modal dismisses immediately. Contractor returns to Quotes tab. (D-02 locked — no waiting in modal)
- Mic button minimum touch target: 80px circle (well above 44px minimum — intentional; primary action of the app)
- Modal background: `colors.dominant` (#ffffff); use `Modal` component with `animationType="slide"`

### ai_processing Row (quotes.tsx QuoteRow)

- Background: `colors.secondary` (#f5f5f5) — subtly different from settled rows
- Left side: quote context (date, customer placeholder or "New job")
- Right side: `ActivityIndicator` (accent color) + "Processing..." in label style, mutedText
- When online: spinner visible — active processing
- When offline/queued: spinner replaced with a static Ionicons `cloud-upload-outline` icon (16px, mutedText) + "Queued" label — no spinner (D-04 locked)
- Row tap while `ai_processing`: no navigation — tap is a no-op (draft not ready yet)
- `accessibilityLabel`: "Quote processing" (online) / "Quote queued" (offline)

### Confidence Badge on LineItemRow (D-05 locked)

Row height rules:
- Clean item (confidence undefined): `minHeight: 56` — unchanged
- Flagged item (confidence 'review' or 'needs_input'): `minHeight: 72` — badge adds ~16px

Left border stripe:
- `borderLeftWidth: 3`
- 'review': `borderLeftColor: '#d97706'`
- 'needs_input': `borderLeftColor: colors.destructive` (#dc2626)
- Clean items: no border

Badge text (sibling `<Text>` below name `<Text>`, same parent `<View flexDirection="column">`):
- 'review': text "Review", fontSize 14, fontWeight '700', color '#b45309' (accessible amber)
- 'needs_input': text "Needs Input", fontSize 14, fontWeight '700', color colors.destructive (#dc2626)
- `numberOfLines={1}` stays on the name Text — badge is a sibling, not a child

Auto-scroll rule: On draft load, if any items have confidence 'needs_input', `FlatList.scrollToIndex()` fires to the first red item. (VOICE-08 locked)

### DraftReadyToast

- Reuses UndoToast container style exactly: bottom 96, left/right md, background #333333, borderRadius 8
- Message: "Your quote is ready — tap to review"
- Entire toast `<Pressable>` navigates to `/draft/${draftId}` — no separate button
- Auto-dismiss: 3000ms (same as UndoToast)
- `accessibilityLiveRegion="polite"` (same as UndoToast)
- `accessibilityRole="button"` on the outer Pressable
- `accessibilityLabel`: "Quote ready. Tap to review."

## Copywriting Contract

| Element | Copy |
|---------|------|
| Voice Quote FAB label | "Voice Quote" |
| Manual FAB label | "Manual" |
| Recording modal title | "Describe the job" |
| Pre-record hint | "Tap to record" |
| Active recording hint | "Tap to stop" |
| Processing handoff text | "Processing..." |
| ai_processing row — online | "Processing..." |
| ai_processing row — offline/queued | "Queued" |
| Draft ready toast | "Your quote is ready — tap to review" |
| Confidence badge: medium tier | "Review" |
| Confidence badge: low tier | "Needs Input" |
| Empty state heading (no quotes) | "No quotes yet" |
| Empty state body | "Tap 'Voice Quote' to describe a job and get a draft in seconds." |
| Mic permission denied | "Microphone access required. Open Settings to allow." |
| Mic permission CTA | "Open Settings" |
| Upload error (sync queue fallback) | "Recording saved. It will upload when you're back online." |
| Whisper failure retry | "Transcription failed. Your recording is saved — tap to retry." |
| GPT-4o timeout | "Mapping timed out. Review the partial draft or tap to retry." |

**Rules:**
- No raw confidence percentages ever shown (VOICE-09 locked)
- "Review" and "Needs Input" are the only confidence labels — no other tier labels
- No error copy mentions internal service names (Whisper, GPT-4o, R2, pg-boss)
- Offline states always reassure: "saved", "queued", "when you're back online"

## Accessibility Contract

| Element | Requirement |
|---------|-------------|
| All FABs | `accessibilityRole="button"`, descriptive `accessibilityLabel` |
| Mic/Stop button | `accessibilityLabel` updates dynamically: "Start recording" / "Stop recording" |
| ai_processing row | `accessibilityLabel`: "Quote processing" or "Quote queued, will upload when online" |
| DraftReadyToast | `accessibilityLiveRegion="polite"` — screen reader announces on appearance |
| Confidence badges | Must not rely on color alone — text label is primary signal (D-05 locked) |
| Left border stripe | Decorative only — not the sole indicator; badge text is always present |
| All touch targets | `minHeight: 44`, `minWidth: 44` minimum — FABs use 56px (D-01 locked) |
| RecordingWaveform | `accessibilityElementsHidden={true}` — decorative animation, not informational |
